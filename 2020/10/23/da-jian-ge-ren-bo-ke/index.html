搭建博客需要node.js的npm和git环境

首先下载npm和git

安装hexo运行环境

  1. npm和git安装完成后的检验是 npm -v 和 node -v

  2. 如图：

安装hexo基本框架

  1. 新建一个文件夹给自己的博客找个小窝

  2. 然后进入新建的文件夹，右击Git Bash Here

  3. //第一条   这是安装hexo的基础框架 , npm安装到全局，熟悉的话可以安装局部
npm install -g hexo

//第二条   这是初始化hexo框架 这个可能会比较慢(要有耐心)
hexo init

//第三条 安装所需要的组件
npm install

//第四条 编译生成静态页面
hexo g

//第五条 启动本地服务
hexo s

  1. 启动后的结果

  2. 然后访问 http://localhost:4000/

hexo命令大全：

hexo n "博客名字" #执行成功后会在根目录下\source\_posts生成一个同名的md文件 
#编写这个文件就可 然后执行 hexo clean hexo g  hexo d 部署到github上去

hexo clean #清除缓存 网页正常情况下可以忽略此条命令
hexo g  #生成静态网页
hexo s  #启动服务预览
hexo d  #开始部署

hexo g == hexo generate#生成
hexo s == hexo server #启动服务预览
hexo d == hexo deploy#部署

github配置个人博客免域名

  1. 注册一个github账户，用邮箱就行

  2. 然后点击new

  4. 我的因为创建过了所以报错

  5. 创建repository的名字一定要是 用户名.github.io,不然访问不到

  6. 之后将code哪里http的git连接复制粘贴到 安装好的hexo的 _config.yml 中

    deploy:
  type: git
  repo: https://github.com/xu196/xu196.github.io.git
  branch: master

    里面 , 运行下面代码在根目录的命令行界面

    hexo clean
hexo g
hexo d
    • 如果上面repo配置的是http的路径hexo d 之后需要输入github的用户名和密码
    • 要是想要无密码push就需要配置ssh的路径，这里不做过多说明，需要自行百度
    • hexo d 成功页面
    • 接下来就可以访问博客了， 就是之前设置的repository名字，我的是xly-star.github.io
    • 接下来就可以尽情的开发自己的博客了
    • 美化博客请看接下来的后续博客！
